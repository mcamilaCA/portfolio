## The Why

Although I had previously coded a portfolio, this time  I had two goals: to learn TypeScript and to complete a full-stack project from start to finish. 

To make it a bit more interesting, I imposed a strict rule for myself: AI use has to be kept to a minimum,  and when used, it has to be as a troubleshooting helper or assistant for specific design questions. I humbly admit that for this to be possible, I had to complete a JavaScript course on Codecademy.
